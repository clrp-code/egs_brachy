get shape volume from a shape input item 

get\+Shape\+Volume takes an EGS\+\_\+\+Input for a shape and returns the volume of the shape. Currently the volume will be calculated automatically for cylinders, spheres and box shapes. Other shapes must specify a volume using the {\itshape shape volume} input key. For example\+: \begin{DoxyVerb}:start shape:
    type = my_new_shape
    input key 1 = 1234
    input key 2= 5678
    shape volume = 123456
:stop shape:
\end{DoxyVerb}


If {\itshape shape volume} is present for a cylinder, sphere or box shape that value will be used and the automatic calculation will be ignored
