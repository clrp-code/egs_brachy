\doxysubsection{Detailed Description}
A class to represent a single phantom for scoring dose in egs\+\_\+brachy. 

A simulation may have an arbitrary number of these phantoms. This class handles scoring both tracklength and interaction scoring, outputing the top N doses to the console and output to 3ddose files.

Region numbers are all \textquotesingle{}local\textquotesingle{} region numbers. That is the phantom does not know anything about its global region number. egs\+\_\+brachy should convert to local region number before calling any method taking a region number as input.
