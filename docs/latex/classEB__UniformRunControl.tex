Uses \textquotesingle{}watcher\textquotesingle{} jobs to determine if the simulation has finished. 

If the current job is a \textquotesingle{}watcher\textquotesingle{} job, it waits for some time before issuing the signal to recombine all available parallel jobs. These \textquotesingle{}watcher\textquotesingle{} jobs can also produce intermediate results while waiting. If all jobs complete while waiting, the \textquotesingle{}watcher\textquotesingle{} job combines all results and exits.
