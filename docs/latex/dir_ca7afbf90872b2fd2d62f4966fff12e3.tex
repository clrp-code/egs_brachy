CLRP Eye Plaque Database version 2 (CLRP\+\_\+\+EPv2) of egs\+\_\+brachy models for photon-\/emitting sources

This repository contains geometry model files for 8 COMS, 5 BEBIG, and 4 Representative models for Pd-\/103, I-\/125, and Cs-\/131 seeds described and validated in the CLRP\+\_\+\+EPv2 database. The models are to be used with the open-\/source Monte Carlo code egs\+\_\+brachy which is an EGSnrc application available at \href{https://github.com/clrp-code/egs_brachy}{\texttt{https\+://github.\+com/clrp-\/code/egs\+\_\+brachy}}. The CLRP-\/\+TG43v2 database is hosted at a Carleton University website\+: \href{https://physics.carleton.ca/clrp/eye_plaque_v2}{\texttt{https\+://physics.\+carleton.\+ca/clrp/eye\+\_\+plaque\+\_\+v2}} with DOI \href{http://doi.org/10.22215/clrp/EPv2}{\texttt{http\+://doi.\+org/10.\+22215/clrp/\+EPv2}}.
