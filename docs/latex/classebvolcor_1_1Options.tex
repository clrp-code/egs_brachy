\doxysubsection{Detailed Description}
Volume correction initialization helper class. 

\doxylink{classebvolcor_1_1Options}{Options} is a small helper class for parsing a volume correction input item A sample input including both a source sspecific correction and a general purpose correction would look like this\+: \begin{DoxyVerb} :start volume correction:


     :start source volume correction:
         correction type = correct # correct, none, zero dose
         density of random points (cm^-3) = 1E8

         :start shape:

             type = sphere
             radius = 0.11
             midpoint = 0 0 0

         :stop shape:

     :stop source volume correction:

     :start extra volume correction:

         correction type = correct # correct, none, zero dose
         density of random points (cm^-3) = 1E5

         :start shape:

             type = box
             box size = 8
             midpoint = 0 0 0

         :stop shape:

     :stop extra volume correction:

 :stop volume correction:
\end{DoxyVerb}


An EGS\+\_\+\+RNG input can also optionally be included in either of the volume correction inputs to use something other than the EGSnrc default RNG.
