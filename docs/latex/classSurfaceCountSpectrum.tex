\doxysubsection{Detailed Description}
A class for scoring a histogram of the number of particles escaping a source geometry. 

Sample input\+: \begin{DoxyVerb}:start spectrum scoring:
    type = surface count
    particle type = photon
    minimum energy = 0.001
    maximum energy = 1.00
    number of bins = 1000
    output format = xmgr
:stop spectrum scoring:
\end{DoxyVerb}
