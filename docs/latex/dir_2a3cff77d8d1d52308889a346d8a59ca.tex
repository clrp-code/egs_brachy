CLRP TG43v2 low-\/energy seeds

This repository contains geometry model files for 40 low-\/energy brachytherapy seeds described and benchmarked in the CLRP TG-\/43 dosimetry parameter database. The models are to be used with the open source Monte Carlo code egs\+\_\+brachy which is an EGSnrc application available at \href{https://github.com/clrp-code/egs_brachy}{\texttt{https\+://github.\+com/clrp-\/code/egs\+\_\+brachy}}.

The CLRP-\/\+TG43v2 database is hosted at a Carleton University website (\href{https://physics.carleton.ca/clrp/egs_brachy/seed_database_v2}{\texttt{https\+://physics.\+carleton.\+ca/clrp/egs\+\_\+brachy/seed\+\_\+database\+\_\+v2}}) with DOI doi.\+org/10.22215/clrp/tg43v2
